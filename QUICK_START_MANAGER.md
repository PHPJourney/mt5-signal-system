# MT5 Manager - 快速启动指南

## 🚀 3 分钟快速开始

### 第一步：启动管理面板

**Windows 用户：**
```bash
双击 start_manager.bat
```

**macOS/Linux 用户：**
```bash
./start_manager.sh
```

### 第二步：配置 Master 服务器

1. 点击 **"⚙️ Master 配置"** 选项卡
2. 填写 MQTT 配置：
   - Broker 地址：`localhost`（或您的 MQTT 服务器地址）
   - 端口：`1883`
   - 用户名/密码：（如果有的话）
3. 填写 MT5 账户信息：
   - 账户登录号
   - 密码
   - 服务器名称
4. 配置信号参数：
   - 交易品种：`EURUSD,GBPUSD,USDJPY`
   - 最大持仓数：`5`
   - 手数：`0.01`
5. 点击 **"💾 保存 Master 配置"**

### 第三步：配置 Slave 服务器

1. 点击 **"⚙️ Slave 配置"** 选项卡
2. 填写 MQTT 配置（与 Master 相同）
3. 填写 MT5 账户信息（Slave 的账户）
4. 设置订阅的 Master ID：`master_001`
5. 配置风险管理参数
6. 点击 **"💾 保存 Slave 配置"**

### 第四步：启动服务

1. 回到 **"📊 仪表板"** 选项卡
2. 点击 **"启动 Master"** 按钮
3. 等待状态变为绿色 "● 运行中"
4. 点击 **"启动 Slave"** 按钮
5. 等待状态变为绿色 "● 运行中"

### 第五步：监控运行状态

- 在 **"📊 仪表板"** 查看实时统计
- 在 **"📡 实时监控"** 查看连接状态和信号历史
- 在 **"📋 日志查看"** 查看详细日志

## 🔧 故障排除

### 问题 1：无法启动服务
**解决方案：**
1. 检查配置文件是否正确
2. 确认 MQTT Broker 正在运行
3. 查看日志文件了解错误详情

### 问题 2：界面显示异常
**解决方案：**
```bash
# 安装美化主题
pip install ttkbootstrap
```

### 问题 3：依赖缺失
**解决方案：**
```bash
# 安装必要依赖
pip install psutil
```

## 💡 提示

1. **首次使用**：建议先测试 MQTT 连接是否正常
2. **配置备份**：定期备份 config 目录下的配置文件
3. **日志管理**：定期清理旧日志，避免文件过大
4. **安全注意**：配置文件中包含敏感信息，注意保护
