## 🎯 已实现的核心功能

### ✅ 1. 实时信号传输
- 基于 MQTT 协议的低延迟通信
- 支持 QoS 1 确保消息可靠送达
- 自动重连机制
- 心跳检测

### ✅ 2. 灵活跟单倍数
- 支持任意倍数设置 (0.1x, 0.5x, 1x, 2x, 5x 等)
- 自动根据经纪商要求标准化手数
- 最小/最大手数限制
- 手数步长自动对齐

### ✅ 3. 反向交易功能
- 一键启用反向跟单
- 买入变卖出,卖出变买入
- 适合对冲和套利策略
- 不影响主服务器逻辑

### ✅ 4. 完整市场映射
- 支持不同经纪商的品种名称映射
- 智能匹配常见变体:
  - XAUUSD ↔ GOLD
  - BTCUSD ↔ BITCOIN
  - US30 ↔ DJI30
  - NAS100 ↔ NDX100
- 动态刷新可用品种列表
- 点差检查和品种信息查询

### ✅ 5. 全面风险管理
- **最大手数限制**: 防止过度交易
- **每日亏损限制**: 达到限额自动停止
- **点差过滤**: 点差过大不执行交易
- **实时风险监控**: 日志记录和状态跟踪

## 🚀 Windows 使用方法

### 方法1: 一键部署 (最简单)
```
双击运行: deploy.bat
```
会自动完成:
- ✅ 检查Python环境
- ✅ 安装所有依赖
- ✅ 引导MQTT安装
- ✅ 打开配置面板
- ✅ 启动服务

### 方法2: 分步安装

**步骤1: 安装依赖**
```
双击: install.bat
```

**步骤2: 配置系统**
```
双击: config_panel.bat
```
在图形界面中配置:
- MQTT连接信息
- 跟单倍数
- 反向交易开关
- 风险管理参数
- 品种映射关系

**步骤3: 启动服务**
```
双击: start_master.bat   (主服务器)
双击: start_slave.bat    (从服务器)
```

## 🎨 配置面板功能

运行 `config_panel.bat` 或 `python config_panel.py` 打开图形化配置界面:

### 选项卡1: 主服务器配置
- MQTT Broker地址和端口
- 认证信息 (用户名/密码)
- 客户端ID
- 信号检测间隔
- 是否包含持仓/挂单信息

### 选项卡2: 从服务器配置
- MQTT连接配置
- **跟单倍数** (核心功能)
- **反向交易开关** (核心功能)
- 最大/最小手数限制
- 滑点设置
- 每日最大亏损限制
- 点差过滤设置

### 选项卡3: 品种映射
- 可视化添加/编辑/删除映射
- 树形列表显示所有映射关系
- 支持批量操作

**底部按钮:**
- 💾 保存所有配置
- 🔄 重置为默认
- 🔌 测试MQTT连接
